    Результатом работы программы является новый файл с объединенным содержимым
входных файлов, отсортированным по возрастанию или убыванию путем сортировки
слиянием. Программа сортирует только текстовые (String) или числовые (Integer)
данные. Если содержимое исходных файлов не позволяет произвести сортировку
слиянием (например, нарушен порядок сортировки), производится частичная
сортировка (сортируются только валидные данные). Выходной файл содержит
отсортированные данные даже в случае ошибок, однако возможна потеря ошибочных
данных.

    Запуск программы выполняется из коммандной строки вводом команды типа:
sort-it.py [-h] [-a | -d] (-i | -s) output_file input_files [input_files ...],
где:
    Обязательные аргументы:
      output_file  обязательный атрибут, путь к выходному файлу;
      input_files  обязательный атрибут, пути к входным файлам. Требуется путь
    минимум одного файла.

    Необязательные аргументы:
      -h, --help   Показывает это сообщение и закрывает программу;
      -a           необязательный атрибут, используйте для сортировки по
    возрастанию (по умолчанию);
      -d           необязательный атрибут, используйте для сортировки по
    убыванию;
      -i           обязательный атрибут, используйте для сортировки числовых
    данных;
      -s           обязательный атрибут, используйте для сортировки текстовых
    данных;

При разработке программы принимались следующие допущения:
    1) Предполагается, что данные во входных файлах отсортированы
соответственно целевой сортировке. То есть если выходной файл должен быть
отсортирован по убыванию, то и входные файлы будем считать отсортированными по
убыванию;
    2) Предполагается, что таблица символов входящих текстовых строк - Unicode,
а кодировка соответствует кодировке операционной системы на которой запускается
программа;
    3) Предполагается, что имеем дело только с десятичными целыми числами.
    4) При неправильной сортировке содержимого входных файлов программа будет
сортировать только корректно отсортированные данные;
    5) Входные файлы после сортировки остаются неизменными.

    Пример команды запуска:
python sort-it.py -s -d output testdata\reversed\test1sl testdata\reversed\test2sl testdata\reversed\test3i testdata\reversed\test4i testdata\reversed\test5i testdata\reversed\test6sc testdata\reversed\test7sc testdata\\reversed\\test8sc testdata\\reversed\\test9u

    Так же к проекту приложен скрипт собранный в исполняемый файл. Пример использования:
sort-it.exe -s -d output testdata\reversed\test1sl testdata\reversed\test2sl testdata\reversed\test3i testdata\reversed\test4i testdata\reversed\test5i testdata\reversed\test6sc testdata\reversed\test7sc testdata\\reversed\\test8sc testdata\\reversed\\test9u

Программа sort-it.py успешно завершила работу. Отстортированные данные сохранены в файл {полный путь к файлу output}.
